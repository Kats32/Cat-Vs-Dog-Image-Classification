# Cat-Vs-Dog-Image-Classification

This project performs binary image classification in a Kaggle Cats Vs Dogs dataset using a fine-tuned MobileNetV3 Small model.

## Project Objective

To analyze and classify images from the dataset into two classes:

- Cat
- Dog

## Model

-**Model:** MobileNet V3 Small
-**Architecture type:** Convolutional Neural Network (CNN)
-**Task:** Binary Image Classification

MobileNetV3 is a lightweight CNN architecture designed for efficient image classification. These models are often adapted and applied to object detection and semantic segmentation tasks.
